Key Features

Private by Default

Every transaction is automatically private. No settings, no opt-in, no compromises.

Quantum Resistant

Hash-based cryptography designed to withstand quantum-level attacks today and tomorrow.

Proof of Work

GPU-mineable consensus. Anyone can secure the network and earn XNT rewards.

Smart Contracts

Program with full privacy via Triton VM. Inputs, outputs, and logic stay hidden.

Community Governed

No CEO, no VC backing, 100% open source. Neptune is built by and for the community.

Fair Economics

No presale, no insider allocation. 100% of supply beyond premine goes to miners.
